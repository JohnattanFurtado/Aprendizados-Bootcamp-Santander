promises:
objeto de processamento assincrono
inicialmente valor desconhecido, pode ser resolvido ou rejeitado
tem 3 estados: pendente, completado ou rejeitado.

async/await
async definda para funcoes assincronas
pra resolver.
funcao assincronas precisam dessas duas palavras chaves.
Quando vai resolver uma promise, precisa dizer que a funcao vai ser assincrona, para que possa usar a palavra await, que é uma palavra criada especificamente para lidar com promise. Await vai parar o codigo.
funcao assin nao pode so chamar, precisa passar o await.
